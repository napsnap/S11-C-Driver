S11-C-Driver

Generic C driver for Sensair Sunrise CO2 sensor (S11)

Introduction

The Sensair Sunrise CO2 sensor is a low power sensor suitable for battery operated equipment. It has two hardware interfaces, I2C and modbus(via UART). This driver supports the I2C mode exclusively.

All sensor features are supported including:

  • Continuous measurement mode
  • Single measurement mode
  • Sensor calibration
  • Sensor configuration
  • Sensor info

Compatability & Integration

This driver is intended for microcontroller use, but can be used by any OS with a C compiler and an I2C peripheral or interface. The driver takes care of all logic, parsing and state changes. All the user has to provide are callbacks for I2C read and write functions.
