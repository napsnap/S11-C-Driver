Calibration Command Register (CAL_CMD) - Calibration types

Defines which calibration routine to perform

#define S11_CAL_CMD_FACT_CAL   (0x7C02)
 Restore factory calibration.
 
#define S11_CAL_CMD_ABC_CAL   (0x7C03)
 Forced ABC calibration.
 
#define S11_CAL_CMD_TARGET_CAL   (0x7C05)
 Target value calibration.
 
#define S11_CAL_CMD_BKG_CAL   (0x7C06)
 Background calibration [400ppm].
 
#define S11_CAL_CMD_ZERO_CAL   (0x7C07)
 Restore factory calibration [0ppm].
 

Detailed Description

API for (re-)calibration of sensor

Usage:

  • Select calibration method by setting bits in s11_cal.cal_cmd register
  • Set calibration target value in ppm in s11_cal.cal_target register (if performing a target calibration)
  • Call s11_calibrate() to perform calibration
  • Call s11_get_cal_status() to retrieve calibration status
  • Read calibration results from s11_cal.cal_status register
